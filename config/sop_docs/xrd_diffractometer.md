## 二、样品制备要求

### 2.1 粉末样品
1. 研磨至粒度<10μm（手感无颗粒感）
2. 取适量粉末填入样品槽
3. 用载玻片压平，确保表面平整无裂纹
4. 样品面与样品架表面齐平
5. 标注样品名称和测试条件

### 2.2 块体样品
1. 切割至合适尺寸（<20mm×20mm×5mm）
2. 测试面抛光至镜面
3. 用橡皮泥固定于样品架上
4. 确保测试面与基准面平行

### 2.3 薄膜样品
1. 基片尺寸<25mm×25mm
2. 薄膜厚度>50nm（过薄信号弱）
3. 固定于薄膜样品架上
4. 进行掠入射对准

## 三、操作流程

### 3.1 开机
1. 检查冷却循环水（水温20±2°C）
2. 开启X射线发生器电源
3. 逐步升压：20kV→30kV→40kV（每步等待1分钟）
4. 升流至40mA
5. 预热30分钟后开始测试

### 3.2 测试参数设置
- 扫描范围：通常 5°~90° (2θ)
- 步长：0.02° （普通）/ 0.01° （高分辨）
- 每步时间：0.1s~2s
- 狭缝：可变发散狭缝

### 3.3 数据采集
1. 装入样品
2. 设置扫描参数
3. 启动扫描
4. 监控X射线管功率（<1.6kW）
5. 保存原始数据（.raw格式）

### 3.4 关机
1. 逐步降压降流：40mA→20mA→10mA→0
2. 40kV→20kV→0（每步等待1分钟）
3. 等待X射线管冷却（>15分钟）
4. 关闭冷却水
5. 填写使用记录

## 四、安全注意事项

⚠️ **X射线辐射安全（关键）**：
1. 操作人员须持有XRD操作资质
2. 测试期间严禁打开防护门
3. 定期检查辐射监测仪（每月一次）
4. 孕妇禁止进入X射线实验室
5. 发现辐射泄漏报警立即按下急停按钮
6. 每年参加辐射安全培训

## 五、常见问题

### Q1: 衍射峰强度低
- 样品量不足 → 增加填充量
- 样品荧光效应 → 更换波长或使用单色器
- 结晶度差 → 延长扫描时间

### Q2: 峰位偏移
- 样品表面未齐平 → 重新装样
- 仪器零点漂移 → 用标准硅粉校准

### Q3: 背底过高
- 非晶相含量高 → 正常现象
- 荧光效应 → 使用能量鉴别器

## 六、预约规则

- 单次最长预约：4小时
- 需持有XRD操作证书（一级以上）
- 粉末样品测试不超过2小时/次
- 高温附件需提前24小时预约
