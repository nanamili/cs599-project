## 二、样品前处理

### 2.1 水质样品
1. 采集后用0.45μm滤膜过滤
2. 加硝酸酸化至pH<2
3. 4°C冷藏保存，48小时内分析

### 2.2 固体样品
1. 称取0.1-0.5g样品于消解罐中
2. 加入5mL HNO₃ + 2mL H₂O₂
3. 微波消解：180°C，30分钟
4. 冷却后定容至50mL
5. 稀释至合适浓度（TDS<0.2%）

### 2.3 生物样品
1. 冷冻干燥后研磨成粉
2. 称取0.2g，加入5mL HNO₃预消解过夜
3. 微波消解：190°C，40分钟
4. 赶酸至1mL，定容至25mL

## 三、仪器操作流程

### 3.1 开机与调谐
1. 检查氩气压力（>0.6MPa）
2. 开启冷却循环水
3. 启动真空系统，等待真空度<1e-5 Pa
4. 等离子体点火（RF功率1550W）
5. 使用1ppb调谐液进行仪器调谐：
   - 灵敏度：Li>3000 cps, Y>10000 cps, Tl>6000 cps
   - 氧化物比率：CeO/Ce<1.5%
   - 双电荷比率：Ba++/Ba<3%

### 3.2 分析方法设置
1. 选择分析模式：定量/半定量/同位素比率
2. 设置积分时间：0.3-1.0s/同位素
3. 设置重复次数：3次
4. 配置标准曲线：至少5个浓度点

### 3.3 样品分析
1. 空白溶液冲洗（2% HNO₃）2分钟
2. 标准溶液分析，建立校准曲线（R²>0.999）
3. 样品分析，每10个样品插入一个QC标准
4. 数据采集与保存

### 3.4 关机
1. 用2% HNO₃冲洗进样系统5分钟
2. 用去离子水冲洗5分钟
3. 熄灭等离子体
4. 松开蠕动泵管
5. 等待真空系统关闭（约30分钟）
6. 关闭冷却水

## 四、安全注意事项

⚠️ **重要**：
1. 操作人员须取得ICP-MS操作资质
2. 消解操作须在通风橱中进行
3. 高氯酸与有机物混合有爆炸风险，禁止使用
4. HF使用需在PTFE器皿中进行
5. 废液分类收集，交由专业公司处理

## 五、常见问题

### Q1: 灵敏度下降
- 清洗锥口、更换采样锥
- 检查蠕动泵管弹性
- 更换雾化器

### Q2: 氧化物比率偏高
- 优化雾化气流速
- 检查冷却水温度
- 降低RF功率

## 六、预约规则

- 单次最长预约：8小时
- 每周一上午为维护时段，不开放预约
- 样品含HF需提前告知管理员
- 超时使用按2倍机时费计算
