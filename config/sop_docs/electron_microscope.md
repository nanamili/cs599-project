## 二、样品制备要求

### 2.1 粉末样品
1. 取少量粉末（约1mg）分散于无水乙醇中
2. 超声分散5-10分钟，避免团聚
3. 用滴管取1-2滴悬浊液，滴于碳支持膜铜网上
4. 自然晾干或红外灯烘干（温度不超过50°C）
5. 确认铜网干燥后装入样品杆

### 2.2 块体样品
1. 机械减薄至100μm以下
2. 使用凹坑仪减薄中心区域至20-30μm
3. 离子减薄直至穿孔
4. 样品转移至铜网固定

### 2.3 生物样品
1. 戊二醛固定2小时（4°C）
2. 磷酸缓冲液清洗3次，每次10分钟
3. 锇酸后固定1小时
4. 梯度乙醇脱水（30%、50%、70%、90%、100%）
5. 环氧树脂包埋
6. 超薄切片（厚度60-80nm）
7. 醋酸铀和柠檬酸铅染色

## 三、操作流程

### 3.1 开机步骤
1. 检查冷却循环水温度和流量（水温20±2°C，流量>3.5L/min）
2. 检查真空系统状态（IP1<1e-4 Pa, IP2<1e-5 Pa）
3. 开启电子枪（加速电压200kV）
4. 等待灯丝电流稳定（约15分钟）
5. 进行合轴调整

### 3.2 合轴调整
1. 电压中心调整
2. 电流中心调整
3. 聚光镜光阑对中
4. 物镜光阑对中
5. 消像散调整

### 3.3 图像采集
1. 选择放大倍数（建议从低倍开始：5k→20k→50k→100k）
2. 聚焦调整（使用Z轴微调）
3. 选择曝光时间（0.5-2秒）
4. 保存图像（支持TIFF和DM3格式）

### 3.4 关机步骤
1. 取出样品杆
2. 降低加速电压至0
3. 关闭电子枪灯丝电流
4. 等待真空恢复（约5分钟）
5. 关闭循环水（再运行15分钟降温）
6. 记录使用日志

## 四、安全注意事项

⚠️ **重要安全事项**：
1. 操作人员必须持有TEM操作证书（二级以上）
2. 未经培训人员禁止独立操作
3. 液氮补充时需佩戴防冻手套和护目镜
4. 发现异常噪音或报警，立即停机并报告管理员
5. 样品必须完全干燥，含水样品会污染真空系统
6. 磁场敏感样品需进行消磁处理

## 五、常见问题与解决方法

### Q1: 图像模糊/漂移
- 原因：样品未固定牢固、热漂移
- 解决：重新固定样品，等待热平衡（10-15分钟）

### Q2: 真空度不达标
- 原因：样品含水、密封圈老化
- 解决：更换干燥样品，检查O型圈

### Q3: 灯丝电流不稳定
- 原因：灯丝老化
- 解决：联系管理员更换灯丝
