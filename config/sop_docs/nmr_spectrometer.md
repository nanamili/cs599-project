## 二、样品制备

### 2.1 液体样品（常规）
1. 称取样品：1H谱 1-5mg，13C谱 10-50mg
2. 溶于0.5-0.7mL氘代试剂（CDCl₃/DMSO-d₆/D₂O等）
3. 溶液应澄清透明，无悬浮颗粒
4. 用移液管转移至NMR样品管（5mm外径）
5. 样品高度：4-5cm
6. 盖上样品管帽，标注编号

### 2.2 特殊样品
- **空气敏感样品**：手套箱中配制，封口膜密封
- **变温实验**：告知管理员温度范围
- **扩散实验**：使用专用梯度探头

### 2.3 常用氘代试剂
| 试剂 | 用途 | 1H残余峰(ppm) | 含水量要求 |
|------|------|:---:|:---:|
| CDCl₃ | 通用 | 7.26 | 无水 |
| DMSO-d₆ | 极性化合物 | 2.50 | 无水 |
| D₂O | 水溶性样品 | 4.79 | — |
| CD₃OD | 含羟基样品 | 3.31 | 无水 |

## 三、操作流程

### 3.1 样品插入
1. 用擦拭纸清洁样品管外壁
2. 放入旋转器（spinner），用深度规调整高度
3. 确认气流正常后放入磁体
4. 等待温度平衡（约2分钟）

### 3.2 锁场与匀场
1. 选择溶剂，执行 lock 命令
2. 观察锁场信号，等待稳定
3. 运行自动匀场（topshim）
4. 检查水峰半高宽（<1Hz 为合格）

### 3.3 测试流程
1. **1H谱**：
   - 脉冲程序：zg30
   - 扫描次数：16-64次
   - 谱宽：20ppm
   - 采集时间：1-3分钟

2. **13C谱**：
   - 脉冲程序：zgpg30
   - 扫描次数：1024-4096次
   - 谱宽：240ppm
   - 采集时间：30分钟-2小时

3. **二维谱（COSY/HSQC/HMBC）**：
   - 由管理员协助设置
   - 采集时间：20分钟-数小时

### 3.4 数据导出
1. 傅里叶变换 + 相位校正
2. 定标（TMS δ=0.0）
3. 积分（1H谱）
4. 导出为PDF或ASCII文件

### 3.5 结束
1. 取出样品管
2. 恢复标准参数
3. 填写使用记录

## 四、安全注意事项

⚠️ **超导磁体安全（极其重要）**：
1. 磁体始终处于超导状态（5.2K），**不断电**
2. 严禁携带铁磁性物品靠近磁体（钥匙、手机、手表、剪刀等）
3. 心脏起搏器佩戴者禁止进入磁体间
4. 银行卡/门禁卡会被消磁——放储物柜中
5. 液氦和液氮每周补充，发现液面过低立即通知管理员
6. **Quench 应急**：磁体间有氧浓度监测，听到报警立即撤离

⚠️ **样品安全**：
- 氘代试剂多为有机溶剂，在通风橱中操作
- 废液分类收集，不得倒入下水道

## 五、常见问题

### Q1: 锁场失败
- 氘代试剂不足 → 补充至4cm以上
- 氘代度不够 → 更换新开封试剂
- 磁体漂移 → 联系管理员

### Q2: 水峰太宽
- 匀场不佳 → 重新 topshim
- 样品含顺磁性杂质 → 过滤或更换样品

### Q3: 信噪比差
- 样品浓度太低 → 增加样品量
- 扫描次数不够 → 增加 NS
- 探头调谐 → 运行 atma
